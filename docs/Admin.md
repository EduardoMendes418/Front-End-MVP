# ERP Admin System Documentation

This document outlines the features and functionalities specifically for administrators in the ERP system.

## Admin Authentication

To access the administrative area, you must log in through the admin login page.

-   **URL:** `/admin/login`
-   **Email:** `admin@gmail.com`
-   **Password:** `admin@123`

Upon successful authentication, you will be redirected to the main dashboard.

## Admin Dashboard

The admin dashboard provides a comprehensive overview of the system, including:

-   **Quick Stats:** Key metrics and performance indicators.
-   **Recent Activity:** A log of recent user actions and system events.
-   **Top Products:** A list of the best-selling products.

## Access Control

Administrators have full access to all modules and features of the ERP system, including:

-   **Dashboard:** Full access to all dashboard widgets and data.
-   **Sales:** View, create, edit, and delete sales records.
-   **Inventory:** Manage products, categories, and stock levels.
-   **Customers:** Access and manage customer information.
-   **Reports:** Generate and view all system reports.
-   **Settings:** Configure system settings and user permissions.

Regular users have restricted access and cannot view or interact with administrative features.
